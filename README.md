# Data Analytics Portfolio

This repository contains my data analytics projects demonstrating skills in:

- Web scraping  
- Exploratory Data Analysis (EDA)  
- Data visualization & dashboards  
- SQL data analysis  

Each folder represents a standalone project with its own code, data, and documentation.

## Projects

- **01-web-scraping** – Scraped book data from a website and cleaned it for analysis  
- **02-exploratory-data-analysis** – Performed EDA using Python and pandas  
- **03-dashboard** – Created visual dashboards to communicate insights  
- **04-sentiment-analysis** – Analyzed text data to determine sentiment  
- **05-sql-data-analysis** – Queried and analyzed data using SQL  

## Tools & Technologies

- Python (pandas, matplotlib, seaborn, BeautifulSoup)
- SQL
- Jupyter Notebook
- Git & GitHub

## Goal

This portfolio is part of my preparation for **Data Science internships**, showcasing hands-on projects and real data workflows.
