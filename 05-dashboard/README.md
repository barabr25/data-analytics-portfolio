# Interactive Dashboard — Book Pricing & Sentiment Analysis

This folder contains the interactive Tableau dashboard built from the final processed dataset, combining pricing data and sentiment analysis results.

The dashboard allows users to explore relationships between book prices and sentiment scores through interactive visualizations.

---

## Dashboard Overview

The Tableau dashboard includes three main visual components:

### 1. Book Price vs Sentiment Score
- Scatter plot showing individual books
- X-axis: Sentiment Score (VADER compound score)
- Y-axis: Book Price (£)
- Colored by sentiment label (positive / negative)
- Interactive tooltips display book title, price, and sentiment

### 2. Price Distribution
- Histogram showing the distribution of book prices
- Highlights common price ranges and overall spread
- Updates dynamically based on sentiment filters

### 3. Sentiment Breakdown
- Bar chart displaying the number of books by sentiment category
- Acts as an interactive filter for the entire dashboard

---

## Interactivity
- Clicking a sentiment category (positive or negative) filters all charts
- Hovering over points or bars reveals detailed tooltips
- Designed for intuitive exploration without requiring technical knowledge

---

## Tools Used
- Tableau Public
- Data sourced from Python-based scraping, cleaning, SQL analysis, and sentiment analysis
