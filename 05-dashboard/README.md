## Dashboard

This dashboard visualizes book pricing and sentiment analysis results to explore relationships between price and emotional tone.

### Key Features
- Overview KPIs summarizing price and sentiment
- Price distribution across books
- Sentiment distribution of book descriptions
- Scatter plot showing sentiment score vs. price
- Interactive filters for sentiment and price range

### Purpose
The dashboard provides an interactive way to explore insights derived from web scraping, exploratory data analysis, SQL analysis, and sentiment modeling.
